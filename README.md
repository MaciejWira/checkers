
## About The Project

  
Checkers game for two players on one platform.
  

### Built With

* Vanilla JS / HTML / CSS

* Webpack / CSS Modules / SCSS

  

## Getting Started

  

To get a local copy up and running follow these simple steps.

  

### Installation

  

1. Clone the repo

2. Install NPM packages

```sh

npm install

```

### Start dev server

```

npm run start

```

### Deploy

```

npm run build

```
Working instance: http://surus.usermd.net/checkers
